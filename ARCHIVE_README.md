# Archive: Experimental Lattice Tools

This branch contains experimental and superseded code that is no longer actively
developed but preserved for historical reference.

## Files

### lattice_reconstruction.py (4,257 lines)
- **Purpose**: Previous attempt at lattice reconstruction for small-molecule crystallography
- **Status**: Superseded by index2.py in the smx_findexer branch
- **History**: Developed as an early approach to unit cell determination from spot pairs
- **Note**: This code represents substantial experimental work and may contain useful
  algorithms or insights for future development, but is not part of the current
  production workflow

### angle_heatmap.py (518 lines)
- **Purpose**: Interactive angle heatmap visualization tool
- **Status**: Experimental visualization code
- **History**: Tool for visualizing azimuthal angle distributions in diffraction data
- **Note**: May be useful for future visualization work but not currently integrated
  into standard workflows
